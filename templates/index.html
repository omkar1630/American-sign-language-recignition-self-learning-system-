How SignPro Works

1
Open the Application
User runs the app and webcam starts capturing the hand gesture.
2
Hand Landmark Detection
MediaPipe detects 21 key landmarks on the user's hand using computer vision.
3
Rule-Based Gesture Recognition
System uses conditional rules to recognize ASL alphabets based on landmark positions.
4
Display Recognized Alphabet
The matching alphabet is displayed in real-time on the screen instantly.
